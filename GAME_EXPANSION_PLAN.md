## 2. 产品目标

将当前 demo 升级为一个更完整的低龄英语启蒙小游戏产品，核心方向：

1. **完整 A-Z 内容化**
2. **关卡化 / 闯关化**
3. **中文语音引导**
4. **轻量背景音乐与音效体系**
5. **本地进度保存**
6. **为后续组合训练 / 拼读训练预留架构**

---

## 3. 推荐产品结构

### 大关卡设计

#### 第一大关：字母启蒙（A-Z）
目标：认识字母、图片关联、描字。

#### 第二大关：字母识别强化
目标：大写/小写、顺序、找缺失字母、快速辨认。

#### 第三大关：字母组合训练
目标：简单组合、首字母识别、基础拼词启蒙。

#### 第四大关：综合挑战
目标：混合题型、关卡式复习。

## 4. V1 版本范围（这次优先做）

本次 V1 不追求一次做完整产品，而是先做“可扩展骨架”。

### V1 包含

1. **首页升级**
   - 不再只有单个 Start
   - 增加“开始冒险 / 选关 / 设置”入口

2. **关卡选择页**
   - 展示多个 stage
   - 第一大关至少拆成若干组，例如：
     - A-C
     - D-F
     - G-I
     - J-L
     - M-O
     - P-R
     - S-U
     - V-X
     - Y-Z
   - 前续关卡完成后解锁后续关卡（可先做轻量版）

3. **A-Z 数据结构化**
   - 不要求每个字母立刻都有精美专属插图
   - 先把数据组织成可扩展结构
   - 优先保证前几组完整体验，剩余字母可走通用 fallback

4. **设置系统**
   - 中文语音开关
   - 背景音乐开关
   - 音效开关

5. **中文语音引导（原型版）**
   - 先使用浏览器 Web Speech API
   - 重点覆盖关键节点：
     - 欢迎
     - 进入关卡
     - 展示字母
     - 选择提示
     - 描字提示
     - 过关反馈

6. **本地存档**
   - 用 localStorage 保存：
     - 已完成关卡
     - 当前解锁进度
     - 音频设置

## 5. 为什么 V1 先不一口气做完整 A-Z 美术

原因：

1. 先把**课程骨架和产品结构**搭起来更重要
2. A-Z 如果每个都做高质量 SVG，美术工作量会明显上升
3. 当前更应该优先验证：
   - 关卡结构是否舒服
   - 中文语音引导是否合适
   - 家长/孩子是否喜欢这种学习路径

所以 V1 策略应为：

- **前几组字母用高质量定制内容**
- 其余字母先用可运行的数据结构与基础占位策略
- 后续再逐步补齐每个字母的专属内容

## 6. 中文语音策略

### 目标
让低龄儿童在不依赖识字的情况下也能理解当前任务。

### V1 方案
使用浏览器内置 `speechSynthesis`：

#### 优点
- 无需后端
- 实现快
- 适合原型验证

#### 缺点
- 声音统一性一般
- 中文童声未必足够稳定

### 后续可升级方案
改为预生成 mp3 语音包：
- 声音统一
- 更可控
- 更像正式产品

### V1 中文语音文案建议

#### 欢迎页
- 欢迎来到小猪字母冒险
- 点击开始，我们一起学习字母吧

#### 选关页
- 请选择一个关卡
- 完成前面的关卡，就能解锁后面的内容

#### 字母展示页
- 这是字母 A
- A，A，A
- A 代表苹果

#### 识别页
- 请找出苹果
- 请找到字母 A

#### 描字页
- 请描一描字母 A
- 描完后点击完成

#### 正反馈
- 太棒了
- 你答对了
- 我们继续前进吧

## 7. 背景音乐策略

### V1 方案
采用轻量循环背景音乐机制，但先不强依赖外部音频素材。

V1 可分两步：

#### Step 1
先做好：
- BGM 开关 UI
- BGM 状态保存
- 音频播放架构预留

#### Step 2
再接入实际背景音乐文件：
- 轻快
- 低音量
- 不压住语音

### 推荐风格
- 轻木琴
- 轻钢琴
- 儿童玩具音色
- 温和循环

### 必须要求
- 必须有开关
- 默认音量偏低
- 语音播放时可以适当压低 BGM

## 8. 数据结构建议

当前 ROUNDS 是单一数组，不够支撑多关卡扩展。

建议升级为：

```js
LEVEL_GROUPS = [
  {
    id: 'letters-1',
    title: '第一关：A-C',
    description: '认识 A、B、C',
    unlocked: true,
    rounds: [...]
  },
  {
    id: 'letters-2',
    title: '第二关：D-F',
    description: '认识 D、E、F',
    unlocked: false,
    rounds: [...]
  }
]
```

后续更高层可以再包一层 worlds：

```js
WORLDS = [
  {
    id: 'world-1',
    title: '字母启蒙',
    levels: [...]
  },
  {
    id: 'world-2',
    title: '组合训练',
    levels: [...]
  }
]
```
